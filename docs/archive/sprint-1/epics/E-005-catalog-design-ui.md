# E-005: Catalog & Design UI First Light

## Goal

A landscaper can manage their material catalog through the UI — add, edit, delete, search materials with category filtering. This is a prerequisite for the quote flow (E-003) and lights up the Design area on the dashboard.

## Target

- S.2.2 Material catalog: — → ★★ (0.0 → 4.0 effective min)

## Stories

- **S-012**: Material catalog management UI

## Success Criteria

- Catalog page lists tenant's materials with name, category, unit, price
- Add material flow with all required fields
- Edit material inline
- Delete material with confirmation
- Search by name, filter by category
- S.2.2 scenario passes at ★★
